**Enterprise Network Design**

A comprehensive network infrastructure project implementing VLSM, OSPF routing, VLANs, DHCP, NAT, ACLs, and WLAN for a multi-site corporate environment.



**Project Overview**

Course: 31277 Routing \& Switching Essentials

Institution: University of Technology Sydney (UTS)

Tools Used: Cisco Packet Tracer

Completion Date: 17/10/2025



This project involves designing and implementing a complete enterprise network infrastructure connecting a Branch office and Headquarters (HQ) with multiple LANs, including routing protocols, security configurations, and wireless connectivity.
